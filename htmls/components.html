Welcome to our component showcase! Here you'll find a collection of beautifully designed, accessible UI components built with modern web technologies. Each component is crafted with attention to detail and user experience.

Our components feature smooth animations, dark mode support, full accessibility compliance, and responsive design. They're built with React, TypeScript, and Tailwind CSS for maximum flexibility and performance.

All components are easy to integrate into your projects. Simply copy the component code, adjust the props to match your needs, and you're ready to go. Full documentation and examples are provided for each component.

Accordion / FAQ

What technologies are used in this project? +
This project is built with HTML5, JS, CSS v4, Motion (formerly Framer Motion) for animations.
Are these components accessible? +
Yes! All components are built with accessibility in mind using Radix UI primitives, which provide excellent keyboard navigation, screen reader support, and ARIA attributes out of the box.
Can I use these components in my project? +
Absolutely! These components are designed to be reusable and easily customizable. You can copy the code and adapt it to fit your specific needs and design system.
Do the components support dark mode? +
Yes, all components fully support both light and dark modes. The theme automatically switches based on your system preference or manual selection via the theme toggle.
Are the components responsive? +
All components are fully responsive and optimized for desktop, tablet, and mobile devices. They use modern CSS techniques and Tailwind's responsive utilities for seamless adaptation.
